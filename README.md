# Python--解析种子，得到真实下载地址
 原理很简单，通过QQ旋风模拟登陆并调用其服务器资源从而达到解析种子的效果
## 使用方式
 1. 抓包得到自己属于自己QQ信息的url，然后修改代码
 2. 在代码的js区域中输入自己的QQ信息即可
 3. 在运行过程中输入要解析的种子的名字（将运行程序和种子放在同一目录）
